### 数据库结构

1. 权限表（uni-id-roles）
   | 字段 | 类型 | 必填| 描述 |
   | ---------- | --------- | ----| --------------------------------------|
   | \_id | Object ID | 是 | 系统自动生成的 Id |
   | role_id | String | 是 | 角色唯一标识 |
   | role_name | String | 否 | 角色名，展示用 |
   | permission | Array | 是 | 角色拥有的权限列表 |
   | comment | String | 否 | 备注 |
   | created_date| Timestamp | 是 | 角色创建时间 |
