### 目录结构

```bash
├── cloudfunctions              # 云函数

├── pages                       # 页面
│   │── index                   # 首页
│   └── login                   # 登录页
├── static                      # 静态资源
├── store                       # vuex
├── windows                     # leftWindow，topWindow
├── App.vue
├── main.js
├── mainfest.json
├── pages.json
├── postcss.config.js           # postcss 配置（浏览器兼容性）
└── uni.scss
```
